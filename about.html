Scientific Calculator

Java | Swing Application

A reproduction and expansion of a project from a previous semester. This edition was created in Java, as opposed to C# like its predecessor. Includes basic calculation, trigonometry, hyperbolic trigonometry, and unit conversion to hexadecimal and binary both.

Contact Book

C# | Windows Forms

A contact book, as one would find on their phone. Uses File I/O to save the information of individual contacts into a list of contacts on a specially created file. Features a reminder feature for birthdays - for those you are not particularly close to.

PyGames Snake

Python | Pygames

A recreation of the classic game 'Snake' in Python. Python was learned in the process of coding this game and allowed me to learn how I learn in the process. The wordplay was lost on me until I was releasing the project, to be completely honest.
